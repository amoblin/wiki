Gollum是一款基于Git的Wiki系统，由Github开发，最初在Github中作为仓库的wiki使用，随着发展，Gollum已经独立成一个完善的Wiki系统。

Gollum支持如下格式的轻量级标记语言：

- AsciiDoc
- Creole
- Markdown
- MediaWiki
- Org-mode
- Plain Text
- Pod
- RDoc
- reStructuredText
- Textile
