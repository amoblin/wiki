# Markdown 快速指南

## 标题

```markdown
# 这是一级标题
## 这是二级标题
### 这是三级标题
```

## 强调

语法：

```markdown
**加粗** *斜体* ***加粗斜体***
```

效果： **加粗** *斜体* ***加粗斜体***

## 无序列表

语法：

```markdown
- Red
- Green
- Blue
```

效果：

- Red
- Green
- Blue

## 有序列表

语法：

```markdown
1. Bird
1. McHale
1. Parish
```
效果：

1. Bird
1. McHale
1. Parish

## 链接

语法：

```markdown
[Markdown 语法说明](http://wowubuntu.com/markdown/)
```

效果：

[Markdown 语法说明](http://wowubuntu.com/markdown/)

## 图片

语法：

```markdown
![Alt text](/path/to/img.jpg)
```

效果：

![Alt text](./snow.jpg)

## 表格

语法：

```markdown
| a | b | c |
|---+---+---|
| d | e | f |
```

效果：

| a | b | c |
|---+---+---|
| d | e | f |
